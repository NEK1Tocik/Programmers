Description in Russian (описание на русском):
В файле First.html Вы найдёте форму авторизации на фоне которой будет разноцветный фон (по кнопке "Начать" цвет фона будет меняться, а при нажатии кнопки "Стоп" - изменение цвета фона остановится).
В файле Nikita.html Вы найдёте также форму авторизации только там фон менятся не будет, но при введение логина и пароля вылезет табличка с Вашим логином и паролем.
В файле main.py Вы найдёте функцию, которая возвращает строку (возвращает приветствие).

Description in English:
In the file First.html You will find an authorization form with a multicolored background on the background (by clicking the "Start" button, the background color will change, and when you click the "Stop" button, the background color change will stop).
In the file Nikita.html You will also find an authorization form only there the background will not change, but when you enter your username and password, a sign with your username and password will pop out.
In the file main.py You will find a function that returns a string (returns a greeting).
